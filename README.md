Sportswxy-Brasil – Meu Portal de Pesquisa Esportiva

Olá! Seja bem-vindo ao meu primeiro grande projeto web feito totalmente pelo celular.

Este é um site de pesquisa focado no universo dos esportes, onde você pode buscar informações e dados sobre times e jogadores de diversas modalidades, como *Futebol*, *Basquete*, *Vôlei*, entre outras. A ideia foi criar um local simples e rápido para centralizar essa busca.

Sobre o Desenvolvimento

Este site é a prova de que consegui aplicar os conhecimentos da *Imersão Dev da Alura* e, o mais importante, avançar muito por conta própria.

Utilizei:
* HTML para o corpo do site.
* Css para o design e a responsividade (que deu um bom trabalho, mas consegui!).
* JavaScript para toda a lógica de pesquisa e manipulação dos dados.

A maior parte do código foi desenvolvida de forma independente com ajuda das aulas da imersão. O Gemini me ajudou muito como um assistente de código, tirando dúvidas pontuais e complexas sobre a sintaxe do JavaScript, o que me permitiu focar na solução dos problemas do projeto sem ficar travado.

Estou muito feliz com o resultado, pois consegui colocar no ar uma funcionalidade complexa (a busca multimodal) de forma limpa e eficiente.

Como Acessar

Você pode testar a aplicação online agora mesmo aqui pelo github.

Se quiser dar uma olhada no código na sua máquina, é só clonar o repositório e abrir o `index.html` no seu navegador.

Atenção! Infelizmente a tabela do site não está totalmente correta nem em pontos ou até mesmo os times, pretendo atualizar elas ao decorrer do tempo mas a atualização manual é bem difícil, então pretendo comprar APIs.

Próximos Passos:

O plano é continuar evoluindo este projeto, integrando no futuro APIs de dados reais de esportes para que as informações sejam sempre atualizadas!

Fico à disposição para feedback e para falar sobre desenvolvimento. Este é o meu primeiro passo, e espero que venham muitos mais!
